Meng Wang's Blog
================

Welcome. This site is a personal study blog and project portfolio
maintained by **Meng Wang**, a graduate student at Beihang University
(Hangzhou International Campus).

The blog is built with `Sphinx`_ and the `Furo`_ theme, deployed as
part of the *Remote Development* course. Each post here corresponds to
a piece of coursework, a project write-up, or a study note.

.. _Sphinx: https://www.sphinx-doc.org/
.. _Furo:   https://pradyunsg.me/furo/
